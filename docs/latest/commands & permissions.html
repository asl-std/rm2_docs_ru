Команды & Разрешения

Команды


/rm
Разрешение: recipemanager.command.rm
Псевдонимы: recipemanager, rmhelp		информация о плагине и доступные команды


/rmrecipes <material>:[data]:[amount] [i|r]
Разрешение: recipemanager.command.rmrecipes
Псевдонимы: recipe, recipes		найти рецепты, в которых используется указанный предмет


/rmnext
Разрешение: recipemanager.command.rmnext
Псевдонимы: НЕДОСТУПНО		перейти на следующую страницу в /rmrecipes query


/rmprev
Разрешение: recipemanager.command.rmprev
Псевдонимы: НЕДОСТУПНО		перейти на предыдущую страницу в /rmrecipes query


/rmfinditem <name>
Разрешение: recipemanager.command.rmfinditem
Псевдонимы: finditem		искать товары по названию


/rmcheck
Разрешение: recipemanager.command.rmcheck
Псевдонимы: checkrecipes		имитирует загрузку файлов рецептов для проверки на наличие ошибок


/rmreload
Разрешение: recipemanager.command.rmreload
Псевдонимы: reloadrecipes		перезагрузить рецепты / настройки / книги / и т. д.


/rmreloadbooks
Разрешение: recipemanager.command.rmreloadbooks
Псевдонимы: reloadbooks		перезагружать только книги рецептов


/rmextract [special]
Разрешение: recipemanager.command.rmextract
Псевдонимы: rmimport, importrecipes, extractrecipes		превращает все рецепты из других плагинов или модов в текстовый файл, чтобы разрешить переопределение и редактирование


/rmgetbook <title> [#<volume>]
Разрешение: recipemanager.command.rmgetbook
Псевдонимы: getrecipebook		дает книгу рецептов


/rmbooks
Разрешение: recipemanager.command.rmbooks
Псевдонимы: recipebooks		перечисляет доступные книги рецептов


/rmupdate
Разрешение: recipemanager.command.rmupdate
Псевдонимы: rmcheckupdates		проверяет, доступна ли для плагина более новая версия


/rmcreaterecipe
Разрешение: recipemanager.command.rmcreaterecipe
Псевдонимы: createrecipe		создает рецепт из крайних левых ингредиентов 3x3 в инвентаре и удерживаемого предмета в результате


/rmdebug
Разрешение: recipemanager.command.rmdebug
Псевдонимы: НЕДОСТУПНО		помощь по отладке для проверки разрешений и данных блокировки

Для разрешений флага вы также можете использовать псевдонимы флага, я отфильтровал их из этого списка, потому что он стал бы слишком длинным, но разрешения есть.
Например, recipemanager.flag.modexp и recipemanager.flag.xp оба влияют на один и тот же флаг, флаг @modexp , поскольку 'xp' является псевдонимом для 'modexp'.
